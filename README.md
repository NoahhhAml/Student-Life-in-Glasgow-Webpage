# 🎓 Student Life in Glasgow

A simple HTML website showcasing student life in Glasgow, including top places to visit, university life, and useful links.

This project was created as part of a web development assignment.

---

## 🌍 Live Website

After enabling GitHub Pages, your website will be available at:

https://noahhhaml.github.io/Student-Life-in-Glasgow-Webpage/
---

## 📌 Website Overview

### 🏛 Top Places To Visit
The website highlights some of Glasgow’s most popular attractions:
- Kelvingrove Art Gallery and Museum  
- Riverside Museum of Transport and Travel  
- University of Glasgow  

Each location includes an image and a short description.

### 🎓 Life As a Student
This section shows:
- Campus lectures  
- Student societies  
- Social activities  
- Hackathons  
- University life experience  

### 🔗 Learn More
Includes a direct link to the official University of Glasgow website.
